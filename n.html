⚠️ MIFARE Classic — ограничения Web NFC API
MIFARE Classic (ISO 14443-A) не поддерживается браузерным Web NFC API — браузер читает только NDEF-теги. Для работы с MIFARE Classic используйте ручной ввод UID (считайте UID телефоном через приложение NFC Tools / NFC TagInfo) или введите ID с наклейки на метке. UID отображается в любом NFC-приложении на Android.

💳 NFC / MIFARE карты
📱 Как получить UID карты MIFARE Classic
1. Установите на Android: NFC Tools или NFC TagInfo by NXP
2. Откройте приложение → «Читать» → поднесите карту
3. Найдите строку «Serial number» / «UID» — скопируйте (например: 04:A1:B2:C3)
4. Вставьте UID в поле ниже для привязки к карте в системе
